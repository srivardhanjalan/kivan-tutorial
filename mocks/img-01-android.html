01-prerequisites — the Android nobody warned you was this easy
$ ./setup.sh --android
  ↓ installing openjdk@17 …
  ✓ Android command-line tools
  ↓ installing Android SDK packages (~2 GB — the long part)
  ✓ emulator 'kivan' (Pixel 8)
  ✓ hardware keyboard enabled
  ✓ ANDROID_HOME + JAVA_HOME → ~/.zprofile
$ emulator -avd kivan && npx expo run:android # that's it
